# RECIPES API

An API wrapper built using Ruby on Rails to simplify sending requests to themealdb.com/api.php to retrieve recipes.

## Dependencies

- rest-client
- rspec-rails
- webmock

## Configuration

1. Clone the project
2. Run ```sh
$ bundle install
```
